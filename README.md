# A01: California Housing Boxplot

## Purpose
This script loads the California Housing dataset to generate a boxplot of median income distributions.

## Data Used
* **Dataset:** California Housing dataset (sourced from Scikit-Learn).

## How to Run the Script
To run the script on your computer, follow these commands:

### 1. install the requirements txt file
pip install -r requirements.txt

### 2. change your working directory as needed to the A01 repo
cd path/to/A01

### 3. run the code
python src/boxplot.py

## Expected Output
* A saved image file: `figs/boxplot.png`
